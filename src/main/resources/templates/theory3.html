Что такое производная?

Производная функции — предел отношения приращения функции к приращению её аргумента при стремлении приращения аргумента к нулю, если такой предел существует.

Геометрический смысл производной

Геометрический смысл производной состоит в том, что значение производной функии y=f(x) в точке х равно угловому коээфициенту касательной
к графику функции в точке с абсциссой х.

Физический смысл производной

Физический смысл производной состоит в том, что производная от перемещения по времени есть является мгновенная скорость.

В свою очередь, производная от скорости по времени является ускорением. Следовательно, вторая производная от перемещения по времени есть ускорение.
Eсли производная функции y=f(x) положительна для любого x из интервала X, то функция возрастает на X; Если производная функции y=f(x) отрицательна для любого x из интервала X, то функция убывает на X.
